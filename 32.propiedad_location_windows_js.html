Propiedad location del objeto window

Cuando le asignamos una nueva dirección a la propiedad location del objeto window, el navegador redirecciona a dicha página.

Ejemplo

Supongamos que tenemos un botón que al ser presionado ingrese al periódico alcalorpolitico.com, pero primero muestre una ventana de confirmación, si queremos ingresar al periódico. En caso que el visitante presione el botón afirmativo, redireccionamos al sitio de La Nación, en caso contrario mostramos un mensaje:


Disponemos un control HTML de tipo button y capturamos el evento onClick:

<input type="button" value="alcalorpolitico.com" onclick="abrirSitio()">

La función abrirSitio() muestra la ventana con los botones confirmar y cancelar (recordar que el método confirm del objeto window hace esto en forma automática). Si se presiona el botón confirmar, la función confirm retorna true y por lo tanto se ejecuta el verdadero del if:

if (abrirSitio == true) { window.location='http://alcalorpolitico.com'; }else { document.write('Haz decidido no visitar el sitio de noticias de veracruz'); }
